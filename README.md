# Mini API

A web-based API client — a simplified Postman for creating, organizing, executing, and inspecting HTTP requests. Built with React to practice scalable architecture using Context API and reducers.

[![bIVMR.png](https://s13.gifyu.com/images/bIVMR.png)](https://gifyu.com/image/bIVMR)

## Features

- **Collections** — group and manage requests in folders
- **Request builder** — method, URL, headers, query params, and body
- **Environments** — variables like `{{BASE_URL}}` with per-collection attachment
- **Request execution** — send requests and view formatted responses
- **History** — track executed requests and re-run them

## Getting started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to run the app locally.
